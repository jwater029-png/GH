## 目录结构

- `01_docs`：可扩展的文档目录，包含规则、计划、想法、决策
- `02_frontend`：前端应用代码
- `03_backend`：后端服务与 API
- `04_database`：数据库表结构、迁移、种子数据、备份
- `05_infra`：部署、代理、进程、环境模板
- `06_scripts`：工具与维护脚本
