## 一句话:星网是什么

> **星网 = 一个只属于你的、跨所有 AI 的本地大脑。**
>
> 你用任何 AI 干活,它都把"此刻该知道的你"喂过去;它自己像睡觉一样不断整理、沉淀、忘掉没用的,越用越像你;点开它,你能看见自己是怎么想的。**这个懂你的大脑,归你,不归任何 AI 公司。**

## 二、产品长什么样(最终形态 = 三个部件)

```
        ┌───────────────────────────────────────────────┐
        │  脸:一面镜子                                   │  ← 节点图 + "睡眠"
        │  睡眠让大脑自己整理/沉淀/忘掉没用的(不会变乱麻)│     照出你怎么想
        │  节点图让你看见自己的思维长什么样               │
        ├───────────────────────────────────────────────┤
        │  神经:活注入引擎                               │  ← 护城河①
        │  按你当前在干的活,实时挑该知道的几条,喂给任何AI│     不是死配置
        ├───────────────────────────────────────────────┤
        │  身体:一个本地大脑                             │  ← 价值本体
        │  你的偏好/习惯/做事方式,存你自己电脑的markdown  │     归你
        └───────────────────────────────────────────────┘
        引擎藏在身体底下(记忆原子/快慢双层/关联网/睡眠期),你看不见——
        只看到镜子和"确认"按钮。详见《记忆引擎设计》。
```

> **和三层架构的对照(同一个东西、两种切法,别晕):** 三部件(身体 / 神经 / 脸)是给你一眼看懂的脸;三层(地基 / 主柱 / 灵魂)是内部架构叫法(见 CLAUDE.md 硬规则)。对应关系:**身体 = 主柱** · **脸 = 灵魂** · **神经(活注入)** 是从主柱里单拎出来强调的护城河① · 还有一层 **「地基 = 跨工具同步」** 藏在最底下,必需但非卖点,归到第②步「跨所有 AI」去实现,所以不画进这三部件。

**1. 身体 · 一个本地大脑** —— 你的偏好、习惯、做事规矩,存在你自己电脑的 markdown 文件里,跟着你走、不跟项目走。**这是你的私产,星网哪天没了文件照样能用。**

**2. 神经 · 活注入引擎** —— 按你当前在干的活,实时挑出最相关的几条喂给 AI。不是一份谁都一样的死配置。**这是星网独占的位置:行业标准(AGENTS.md)只做"格式统一",永远不做"按任务挑哪几条"。**

**3. 脸 · 一面镜子** —— 一张节点图,加一个"睡眠"机制:
- **睡眠** = 你不用电脑时,它在后台回放、提炼出你的模式、忘掉没用的。所以它**会减**,不会像 Obsidian 越用越乱。
- **节点图** = 让你看见自己的思维逻辑长什么样,伸手改它。**用得越久,图越像真实的你。**

## 三、它和你那些 AI 的关系(一句话框架)

> **你的 AI(Claude 等)= 海马,负责当下快速记一下;星网 = 新皮层 + 睡眠,负责沉淀、抽象、忘记、归你所有。**

星网不抢 AI 的活,补它结构上不做的那一半——而且是**所有 AI 共用的那块大脑**。它们的自带记忆越强,你的大脑被切得越碎,星网这个"汇成一份、归你"的聚合层就越值钱:**它们不是竞品,是数据源。**

---

## 四、两条护城河(力气全砸这)

**① 活注入** —— 按当前任务实时挑,不是生成死文件。这是标准和那几十个开发者工具都迈不过的坎。

**② 镜子 + 归属** —— 节点图会减(睡眠代谢)+ 大白话 + 归用户。**这是给人看的镜子,不是给机器用的记忆。** 对手(Claude/Cursor 的记忆)结构上不会做"跨厂商 + 归用户 + 给普通人看"——帮用户逃离自家锁定,违背他们的商业利益。

**一条战略红线:不跟 Mem0/Zep/Letta 卷"记忆模型谁更先进"(那些是公开货),只卷"镜子 + 归属"。复杂全藏引擎,极简只留脸。**

## 六、怎么走到最终形态(四步,门槛用能力不用人数)

| 阶段 | 走到哪 | 过关 = 能力指标 |
|---|---|---|
| **现在** · 一个工具跑通 | Claude Code 上:偏好真生效 + 镜子会减 + 你自己离不开 | 写一条偏好按任务挑出真生效;节点会熄灭会内化;你离不开它 |
| 下一步 · 跨所有 AI | 同一个大脑,Cursor/Codex 都读得到;把它们自带的记忆也吸纳进来归你 | 同一份偏好三家都生效、零重写;自带记忆能汇成一份 |
| 再下一步 · 值得付费 | skill 资产接入 + 高级活注入 | 复用价值大到愿持续付费 |
| 最终 · 活的镜子 | 睡眠期全自动、主动照出你的模式 | 自动质量高到你敢信;**默认关闭、你亲手开、只建议不替你做** |

**【批次进度】** 批次1 主柱链条 ✅ · 批次2 灵魂层(节点图 2-①✅,余工作记忆快层/新陈代谢/双向同步)· 批次3 半自动蒸馏 · 批次4 创始人 dogfood。详见执行计划。

## 七、不退让的原则(就这几条)

1. **归你** —— markdown 本地,可读可迁移,星网没了也能用。
2. **中立** —— 不绑任何 AI 厂商,星网站在中间那层,只属于用户。
3. **会减** —— 图必须会忘,遵循奥卡姆剃刀,绝不沦为 Obsidian。
4. **只呈现不接管** —— 镜子照给你看,你自己觉察、你拍板;AI 产出先存"待确认",不自动固化(挡歪镜子)。
5. **不承诺做不到的** —— 说好处(帮/让/顺带),不说承诺(防止/避免/替代/消灭)。
6. **不要饭式讨好** —— 门槛用能力指标,只解决客观痛点,不张望谁鼓掌。
7. **不卷模型先进度,只卷镜子 + 归属** —— 复杂藏引擎,极简留脸。
