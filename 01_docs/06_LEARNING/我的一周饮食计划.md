# 我的一周饮食计划

> **为谁定制:** 身高183 / 体重107斤(BMI 14.4偏轻) / 在星巴克站班7-8小时 / 目标=健康增重、稳住能量、消下午脑雾。
> **对标标准:** WHO + 美国膳食指南(蛋白1.6-2.2g/kg增肌区间、蔬果≥400g/天、游离糖<10%)。
> **核心目标:** ①每天蛋白≈90g(达标) ②总热量盈余(增重的命) ③每顿蛋白+油配着碳水吃(防血糖过山车) ④补够蔬果。

---

## 三条铁律(比任何菜单都重要)

1. **米饭绝不空口吃** —— 永远配着蛋/肉/菜,让血糖平稳(防下午脑雾)。
2. **油放够、坚果管够** —— 你太瘦,油和坚果是最省事的增重热量,别清淡。
3. **每周称一次体重** —— 这是唯一的验收标准。往上走=对了;不动=吃得还不够,加量。

---

## 每天固定"地基"(雷打不动,7天都一样)

| 时段 | 吃什么 | 作用 |
|---|---|---|
| 🌅 早餐 | 鸡蛋2个 + 牛奶/豆浆1杯 + 主食(粥/包子/面包) + 1个水果(香蕉最佳) | 定一天血糖基调,黄金窗口供能 |
| ☕ 上班加餐 | 牛奶1盒(店里有) + 水煮蛋1-2个 + 一把花生 | 顶住站班消耗,不烧自己储备 |
| 🍚 午餐 | 米饭1.5-2碗 + 【当天主菜】 + 青菜一份 + 油放够 | 主蛋白+热量主力 |
| 🌙 晚餐 | 米饭1-1.5碗 + 蛋或剩的主菜 + 青菜一份 | 补足全天蛋白 |
| 🌰 睡前(可选) | 一把坚果 或 一杯奶 | 增重加餐,不饿着睡 |

**光这个地基:鸡蛋4-5个 + 牛奶 + 花生 + 米饭3-4碗 + 青菜 + 水果 ≈ 蛋白55g打底。**

## 还要观察的健康信号

- **下午四五点脑雾** → 有没有减轻?(吃对了会减轻)
- **体重** → 每周称,往上走才算成功
- **下巴/嘴周痘** → 先停甜巧克力一周,看好转没;牛奶先留着观察
- **大便** → 保持成型(说明肠胃接得住)
